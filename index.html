Présentation de la session

Depuis l’arrivée de HTML5 et de ses nouveaux éléments, les sites et applications web peuvent profiter de grandes avancées pour rendre ce média attractif : audio, vidéo et Canvas pour disposer de fonctionnalités de dessin en 2D et 3D dans le navigateur. Par conséquent, il est essentiel de pouvoir connaître : Les fonctionnalités globales de canvas Les navigateurs supportant le canvas L’intégration de canvas dans une page web HTML 5 Les tracés graphiques de base (ligne, rectangle, arc) Les tracés de texte .

L’intégration de canvas dans une page web

On va commencer par l'intégration de canvas dans une page et la récuperation de l'objet dans javascript avec “document.getElementById” et puis avec “getContext” 
                     
<!DOCTYPE html>

                     
<html lang="fr">

                        
<head>

                             
<title>
Utiliser le canvas 
                             
</title>
 
                        
</head>

                         
<body>
<p>
Voici mon premier canvas 
</p>

                              
<canvas id="moncanvas" width="200" height="200">
 
                              
Votre navigateur ne supporte pas canvas !
 
                              
</canvas>

                              
<script>

                              
var canvas = document.getElementById('monCanvas'); 

                              
var ctx = canvas.getContext('2d');
         
                              
</script>

                        
</body>

                      
</html>

document.getElementById() : navigation dans le DOM

canvas.getContext('2d ou 3d'); : le contexte sert à expliciter quel contexte de dessin nous allons utiliser pour l’élément que l’on a sélectionné.

Les tracés graphiques de base

Passons maintenant dans le vif du sujet , les étapes d'un tracé et les formes quelconques (ligne,rectangle,arc)
                            
<!DOCTYPE html>

                            
<html lang="fr">

                               
<head>

                                    
<title>
Utiliser le canvas 
                                    
</title>
 
                               
</head>

                                
<body>
<p>
Voici mon premier canvas 
</p>

                                     
<canvas id="moncanvas" width="200" height="200">
 
                                     
Votre navigateur ne supporte pas canvas !
 
                                     
</canvas>

                                     
<script>

                                     
var canvas = document.getElementById('monCanvas'); 

                                     
var ctx = canvas.getContext('2d');
  
                                     
ctx.beginPath();

                                     
ctx.moveTo(50, 100);

                                     
ctx.lineTo(200, 200);
 
                                     
ctx.closePath();
     
                                     
ctx.beginPath();
 
                                     
ctx.rect(200, 50, 220, 140);
 
                                     
ctx.closePath();
   
                                     
ctx.beginPath();
 
                                     
ctx.fillRect(200, 50, 220, 140);
 
                                     
ctx.closePath();

                                     
ctx.beginPath();
 
                                     
ctx.arc(180,130,15,0,Math.PI*2,false);
 
                                     
ctx.closePath();
    
                                     
</script>

                               
</body>

                             
</html>
beginPath(); Initialisation
moveTo(x,y); Point de départ
lineTo(x,y); Point d’arrivée
closePath(); Clôture
ctx.fillRect(x, y, height, width); tracer un rectangle en indiquant ses coordonnées
arc(x ,y ,rayon ,startAngle ,endAngle ,sensAntiHoraire ) |x,y : coordonnées du centre / endAngle : en radiant avec Math.PI / sensAntiHoraire : True ou false

Les tracés de texte

Dans ce chapitre , nous allons écrire un texte en utilisant les méthodes fillText et strokeText
                            
<!DOCTYPE html>

                            
<html lang="fr">

                               
<head>

                                    
<title>
Utiliser le canvas 
                                    
</title>
 
                               
</head>

                                
<body>
<p>
Voici mon premier canvas 
</p>

                                     
<canvas id="moncanvas" width="200" height="200">
 
                                     
Votre navigateur ne supporte pas canvas !
 
                                     
</canvas>

                                     
<script>

                                     
var canvas = document.getElementById('monCanvas'); 

                                     
var ctx = canvas.getContext('2d');
  
                                     
ctx.beginPath();

                                     
ctx.fillText("Hello World !", 50, 50);

                                     
ctx.strokeText("Hello World",150, 150);

                                     
ctx.closePath();
     
                                     
</script>

                               
</body>

                             
</html>
fillText(texte,x,y); : texte en solide
strokeText(texte,x,y) : contour du texte
